Sistema Escolar

Gerenciamento de Notas e Turmas

Descrição do Projeto

Desenvolvi um sistema escolar que permite a interação entre alunos, professores e a secretaria, com funcionalidades específicas para cada grupo.

Tecnologias Utilizadas

  • HTML
  • CSS
  • JavaScript
  • React
  • C# .NET
  • MySQL

Funcionalidades

Tela do Aluno

O aluno pode visualizar suas notas e acompanhar seu desempenho acadêmico.

Tela do Professor

O professor acessa as turmas e lança notas dos alunos, facilitando o gerenciamento das avaliações.

Tela da Secretaria

A secretaria cadastra professores, organiza as turmas e acessa informações sobre as notas dos alunos.
